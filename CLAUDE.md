# 星読みアプリ

## 概要
ブラウザで動く占星術チャートアプリ（シングルHTMLファイル）。
生年月日・出生時刻・出生地を入力してネイタルチャート＋トランジットを表示。

## 現在のバージョン
**v9**（2026-05-06 更新済み）

## 主な機能
- ネイタルチャート（SVG描画）
- トランジットアスペクト（日付変更対応）
- 天体配置・ハウスカスプ・アスペクト一覧テーブル
- CSV保存 / チャート画像保存
- **個人プロフィール保存（localStorage）** ← v7で追加

## ファイル構成
```
astrology-app/
  index.html   ← すべてのコード（HTML/CSS/JS）が1ファイルに統合
```

## 修正履歴
- 2026-05-06: 天王星補正係数バグ修正（0.832→5.31、e≈0.046に正しく対応）。ハウス計算をPorphyryから本物のPlacidus（semi-arc三分割・Newton-Raphson反復）に修正。緯度・経度直接入力フィールドを追加（精度優先時に使用）。
- 2026-05-04: チャート方向バグ修正（時計回り→反時計回り）、ハウス番号ズレ解消、全惑星を地心座標変換に修正（水星・金星のサインズレが主因）

## 技術メモ
- 外部ライブラリなし（Pure JS）
- 天体計算：Swiss Ephemeris相当のロジックをJSで実装
- ハウスシステム：プラシダス
- デザイン：ネイビー×ゴールドのクラシック占星術スタイル
